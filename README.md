# Flappy Bird

A classic Flappy Bird arcade game built with Python and Pygame. Navigate the bird through pipes by timing your jumps to avoid collisions.

## How to Play
- Press **SPACE** to make the bird fly upward
- Avoid hitting the pipes or the top/bottom edges of the screen
- Each pipe you successfully pass adds 1 point to your score
- The game gets faster and more challenging the longer you survive

## Controls
| Key | Action |
|-----|--------|
| SPACE | Jump / Start Game / Restart |

## Requirements
- Python 3.x
- Pygame
